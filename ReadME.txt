This is a Web App for an Image gallery 

All you have to do is search up the image, like you would using google and wait while the images loads

You can also tap on an image to get a fullscreen look at the image and the title of the Image

Api source was from Unsplash and it can't take a lot of load i.e more than 10 requests at A time

You can Toggle the visibility of your favourite images by clicking on the favourite button
